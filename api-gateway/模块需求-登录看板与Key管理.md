## 一、登录与看板模块（01-登录与看板.html）

### 1.1 模块概述

提供两类页面：**登录页**（未认证入口）与**统计看板页**（登录后首页）。看板汇聚全局运营指标，帮助管理员快速掌握平台运行状态。

### 1.2 功能清单

| 编号 | 功能 | 说明 |
|------|------|------|
| F1 | 邮箱+密码登录 | 主登录流程 |
| F2 | 首次登录强制改密 | 初始密码必须更换 |
| F3 | 全局统计卡片 | 4张核心指标卡 |
| F4 | 消耗趋势图 | 7天折线图 |
| F5 | 最近动态流 | 平台操作日志摘要 |
| F6 | 侧边栏导航 | 5个功能入口 |
| F7 | 头像下拉菜单 | 个人资料 + 退出登录 |

### 1.3 登录页 — 详细需求

#### 1.3.1 页面布局

```
┌──────────────────────────┬──────────────────────┐
│                          │                      │
│   品牌区（深色背景）       │     登录表单区        │
│   - 平台名称             │   - "登录" 标题       │
│   - 口号标语             │   - 邮箱输入框        │
│   - 装饰元素             │   - 密码输入框        │
│                          │   - 登录按钮          │
│                          │   - 版权信息          │
└──────────────────────────┴──────────────────────┘
  左侧 1/2                   右侧 1/2
```

#### 1.3.2 表单字段

| 字段名 | 类型 | 必填 | 校验规则 | Placeholder |
|--------|------|------|----------|-------------|
| 邮箱 | email | 是 | 标准邮箱格式 | `you@company.com` |
| 密码 | password | 是 | 非空 | `········` |

#### 1.3.3 交互细节

**密码显示/隐藏**
- 密码输入框右侧有"眼睛"图标按钮
- 点击切换 `type="password"` / `type="text"`
- 图标在睁眼/闭眼之间切换，提供即时视觉反馈

**错误状态**
- 登录失败时，表单上方出现红色提示条
- 提示条含警告图标 + 错误文案
- 邮箱和密码输入框边框变红（`border-red-400`）

**加载状态**
- 点击登录后按钮显示 loading 旋转图标 + "登录中..."
- 按钮 disabled 状态，不可重复点击
- 模拟 1.2s 延迟后进入

**登录成功流程**
1. 模拟验证通过
2. 自动弹出"修改初始密码"弹窗（首次登录）

---

#### 1.3.4 修改密码弹窗

| 字段 | 类型 | 说明 |
|------|------|------|
| 当前密码 | password | 首次登录可跳过验证 |
| 新密码 | password | 8-20位，含字母和数字 |
| 确认新密码 | password | 需与新密码一致 |

**校验规则：**
- 空值校验：点击确认时所有字段非空
- 长度校验：新密码 8-20 位
- 格式校验：必须包含至少1个字母 + 至少1个数字
- 一致性校验：新密码与确认密码必须一致
- 错误时弹窗内显示红色错误提示，不关闭弹窗

**成功流程：**
- 弹窗关闭 → 进入统计看板页

## 二、Key管理模块（02-Key管理.html）

### 2.1 模块概述

Key是平台核心资源。提供 Key 的生成（单个/批量）、全生命周期管理、列表检索与筛选、详情查看等功能。

| K6 | 批量结果展示 | 全屏结果页 + CSV导出 |

| K9 | Key详情 | 弹出详情面板 |
| K10 | Key复制 | 一键复制完整Key到剪贴板 |

### 2.5 单个生成Key — 详细需求

#### 2.5.1 触发方式

点击列表上方【生成Key】按钮 → 弹出右侧抽屉面板

#### 2.5.2 抽屉表单

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| 客户名称 | text + 联想 | 是 | — | 输入≥2字弹联想列表；无匹配时回车自动入库 |
| Key类型 | select | 是 | 测试 | 测试 / 永久 二选一 |
| 额度(元) | number | 条件 | 2.00 | 仅测试Key显示；系统参数动态获取 |
| 备注 | text | 否 | — | 内部标识用 |

**智能联想行为：**
1. 用户输入 ≥2 个字符
2. 系统从客户库匹配，实时显示联想下拉
3. 点击联想项 → 填入并选中
4. 无匹配直接回车 → 新建客户并关联Key

**类型切换联动：**
- 选择"永久Key" → 额度字段隐藏
- 选择"测试Key" → 额度字段出现，默认值2.00

---

#### 2.5.3 生成流程

```
点击[生成Key]
  → 打开右侧抽屉
  → 填写表单
  → 点击确认
  → 校验：客户名称非空
  → 生成1个Key（sk-随机字符串）
  → Key写入列表顶部
  → 关闭抽屉
```

### 2.6 批量生成Key — 详细需求

#### 2.6.1 触发方式

点击列表上方【批量生成Key】按钮 → 弹出右侧抽屉

#### 2.6.2 抽屉表单

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| 客户名称 | text + 联想 | 是 | — | 同单个生成 |
| Key类型 | select | 是 | 测试 | 同单个生成 |
| 生成数量 | number | 是 | 50 | 范围 10-100 |
| 单个额度(元) | number | 条件 | 2.00 | 仅测试Key；所有Key统一额度 |
| 备注 | text | 否 | — | 所有Key共用 |

#### 2.6.3 结果展示页

**触发：** 批量生成成功后 → 关闭抽屉 → 弹出全屏结果页

**结果页内容：**
- 顶部摘要："已成功生成 N 个Key，总额度 ¥X 元"
- Key 清单表格：
  - Key（完整明文）
  - 客户名称
  - 类型
  - 额度
- 【一键导出全部Key信息（CSV）】按钮

**CSV 导出格式：**
- 文件名：`{客户名称}-Key批量-{日期}.csv`
- 列：Key, 客户名称, 类型, 额度
- 无掩码，完整导出
